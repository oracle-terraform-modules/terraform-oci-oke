# Utility Examples

Example configurations for utility features:

| File | Description |
|------|-------------|
| `vars-utilities-drain.auto.tfvars` | Worker pool draining configuration |
| `vars-utilities-ocir.auto.tfvars` | Oracle Container Image Registry (OCIR) secret |
| `vars-utilities-serviceaccount.auto.tfvars` | Kubernetes service account creation |

## Usage

Copy the desired `.auto.tfvars` file(s) to your root module and adjust the values as needed.
